ShellCheck linter plugin for CudaText that integrates with the CudaLint framework.

## Features
* Automatic ShellCheck detection (system PATH or bundled executable)
* Configurable ignore rules via JSON configuration file
* Multi-platform support (Windows, Linux, macOS)
* Severity differentiation (errors vs warnings with color-coded icons)
* Diagnostic logging for troubleshooting
* Support for comment syntax in configuration files (// and #)

## Testing
The plugin has been tested on Windows 11 with both system-installed and bundled ShellCheck executables.
